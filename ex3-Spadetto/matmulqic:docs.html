MATMULQIC

Matmulqic is a simple fortran90 module containing subroutines to compute the products of real matrices, to generate random real matrices in the range[0,1] and to plot such matrices on terminal in a readable way . It is based on the debugqic module.

---- Instances ----

     logical  debug_var :   enables and disables debugging

---- Subroutines ----

set_debug(mybool)

     IN  mybool :   logical, assign the debug_var instance to its value

Assign the debug_var instance to a value.

printmatrix(mat)

     IN  mat :   mat is a real matrix in the form of a fortran real 2d array

Print a matrix on screen

matmul1(aa,bb,result)

     IN  aa :  it is a real matrix in the form of a fortran real 2d array


     IN  bb :  it is a real matrix in the form of a fortran real 2d array


     OUT  result :  it is a real matrix in the form of a fortran real 2d array

Computes the product aa*bb and output result

matmul2(aa,bb,result)

      IN  aa :  it is a real matrix in the form of a fortran real 2d array


      IN  bb :  it is a real matrix in the form of a fortran real 2d array


      OUT  result :  it is a real matrix in the form of a fortran real 2d array

Computes the product aa*bb and output result, the for loops iterating on the result matrix elements are interchanged

matmul3(aa,bb,result)

      IN  aa :  it is a real matrix in the form of a fortran real 2d array


      IN  bb :  it is a real matrix in the form of a fortran real 2d array


      OUT  result :  it is a real matrix in the form of a fortran real 2d array

Computes the product aa*bb and output result, computes first the transpose of aa and then iterates only over the rows of the two matrices

matmul4(aa,bb,result)

     IN  aa :  it is a real matrix in the form of a fortran real 2d array


     IN  bb :  it is a real matrix in the form of a fortran real 2d array


     OUT  result :  it is a real matrix in the form of a fortran real 2d array

Computes the product aa*bb and output result, computes first the transpose of bb and then iterates only over the columns of the two matrices

---- Functions ----

gen_rnd_matr(sizerow,sizecol) result(gen)

     IN  sizerow :  integer, number of rows of the output


     IN  sizecol :  integer, number of rows of the output


     OUT  gen :  it is a real matrix in the form of a fortran real 2d array

Generates a random real matrix in form of a real 2d array, with dimensions sizerow and sizecol, all the elements are in the range [0,1]
